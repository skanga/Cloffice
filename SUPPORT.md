# Support

## Getting Help

For setup and usage help:

- Read [README.md](README.md)
- Check docs in [docs/](docs/)
- Open a GitHub issue with context and logs

## What to Include in Support Requests

- OS and app version
- Steps to reproduce
- Expected result and actual result
- Relevant screenshots/logs
- Whether you are using the built-in internal runtime and which provider or model was selected
